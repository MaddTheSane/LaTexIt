Version 1.16.1

  • Meilleure compatibilité avec des versions anciennes de pdflatex/ghostscript;
  • correction d'un bug dans l'action Automator, qui l'empêchait d'être sauvegardée;
  • correction d'un bug dans la propagation des variables d'environnement pour les scripts personnalisés.

Version 1.16.0

  • Ajout d'une action Automator;
  • meilleure gestion de l'environnement Unix, qui corrige un bug rencontré avec certains configurations;
  • gestion de plusieurs préambules;
  • ajout de la palette "Fichiers supplémentaires";
  • amélioration du découpage automatique des équations;
  • gestion de plusieurs préambules;
  • pour la lisibilité, les icônes des fichiers créés n'ont plus un fond transparent;
  • la détection automatique de la taille des polices par le service est maintenant dotée d'un facteur multiplicatif paramétrable;
  • mise à jour de la traduction Allemande (Peter Dyballa);
  • correction de bugs mineurs.

Version 1.15.0

  • la version pour MacOS 10.3 (Panther) ne sera plus maintenue
  • correction d'une incompatibilité avec le shell de Leopard
  • les services fonctionnent à nouveau sous Leopard
  • gestion plus robuste des raccourcis claviers pour les Services
  • le service "Détecter et créer les équations" comprend maintenant \begin{eqnarray*}...\end{eqnarray*}
  • on peut maintenant importer une bibliothèque provenant de LaTeX Equation Editor
  • les tabulations ne sont plus arbitrairement limitées en nombre
  • LaTeXiT utilise maintenant Sparkle pour les mises à jour
  • la latexisation utilise maintenant le raccourci ⌘T comme TeXShop; ⌘L, ⇧⌘L, ⇧⌘T fonctionnent encore
  • le préambule par défaut utilise maintenant \usepackage[usenames]{color} au lieu de \usepackage{color}
  • nouvelle icône de Jacob Bandes-Storch
