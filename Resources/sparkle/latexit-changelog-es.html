Version 2.1.0 (9 Marzo 2010)

  • temporalmente descartando el soporte para 64 bits por un problema en el AppleTypeServer ya mencionado en la versión 2.0.1;
  • migrando del entorno Eqnarray a Align, que tiene la misma sintaxis pero proporciona mejores resultados;
  • corregido un problema al exportar si al escala no era 100%;
  • mejor retrocompatibilidad con Linkback;
  • mejor traducción a Japonés, gracias a Ryohei Seto;
  • pequeñas mejoras en el interfaz.

Versión 2.0.1 (15 Noviembre 2009)

  • temporalmente sorteando un bug del modo 64-bits, pues por una razón desconocida, hace fallar el AppleTypeServer (con certeza un bug en el propio ATS);
  • arreglados problemas de migración de Library bajo ciertas condiciones;
  • arreglado soporte para Tiger;
  • arreglado soporte para LinkBack;
  • arreglado el botón desactivado usado para aplicar los preámbulos a la biblioteca;
  • mejorados algunos detalles de la interfaz y arreglo de pequeños bugs.

Versión 2.0.0 (4 Noviembre 2009)

  • Revisión de código con muchas mejoras internas para una mayor flexibilidad;
  • plantillas de contenido;
  • el historial y la librería son controlados por Core Data, para mejorar la gestion de los recursos (memoría...);
  • es posible añadir argumentos adicionales a las llamadas a programas subyacentes (pdflatex...);
  • ahora las ventanas de documentos tienen un modo "mini", ala iTunes;
  • recuperada la compatibilidad con procesadores G3;
  • tiempo de arranque mejorado;
  • corregidos problemas en los servicios;
  • corregidos otros problemas menores;
  • traducción al japonés;
  • el desarrollador también ha sido actualizado a la versión 2.0 al casarse con su amada esposa. Cualquier donación para la luna de miel será bienvenida!

Versión 1.16.1

  • better compatilibity with old versions of pdflatex/ghostscript;
  • fixed a bug in the Automator action, that was preventing it to be saved;
  • fixed bug regarding environment variables propagation in custom scripts.

Versión 1.16.0

  • Added an Automator action;
  • better Unix environment handling, which fixes a bug with some configurations;
  • multiple preambles management;
  • added "Additional files" palette;
  • better way to auto-crop equations;
  • multiple preambles management;
  • for the sake of readability, the icons of the created files do not have a transparent background any more;
  • the automatic font size detection through the Service can now use a custom factor;
  • updated German translation (Peter Dyballa);
  • fixed some minor bugs.

Versión 1.15.0

  • la versión MacOS 10.3 (Panther) ya no será mantenida
  • se ha reparado una incompatibilidad con el shell de Leopard
  • los Servicios funcionan de nuevo en Leopard
  • forma más robusta de manejar los atajos de teclado para Servicios
  • el servicio "Detectar y componer ecuaciones" ahora soporta \begin{eqnarray*}...\end{eqnarray*}
  • ahora es posible importar librerías del Editor de Ecuaciones de LaTeX
  • las tabulaciones ya no están limitadas
  • LaTeXiT ahora usa Sparkle para las actualizaciones
  • la latexización ahora es disparada por ⌘T como en TeXShop; ⌘L, ⇧⌘L, ⇧⌘T todavía funcionan
  • el preámbulo por defecto ahora usa \usepackage[usenames]{color} en vez de \usepackage{color}
  • nuevo icono por Jacob Bandes-Storch
