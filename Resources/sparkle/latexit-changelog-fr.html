Version 2.1.0 (9 mars 2010)

  • abandon provisoire du 64 bits à cause de bugs du AppleTypeServer déjà cités dans la version 2.0.1;
  • migration du mode EqnArray à Align, qui s'utilise de la même façon mais donne de meilleurs résultats;
  • correction d'un bug d'export si l'échelle n'était pas 100%;
  • meilleur rétro-compatibilité avec LinkBack;
  • amélioration de la traduction en Japonais, grâce à Ryohei Seto;
  • raffinements d'interface.

Version 2.0.1 (15 novembre 2009)

  • contournement provisoire d'un bug en 64 bits pour le moment, car pour une raison inconnue, le AppleTypeServer semble incompatible avec et plante (sans doute un bug dans le ATS lui-même);
  • correction de problèmes de migration de la bibliothèque sous certains conditions;
  • correction du support de Tiger;
  • correction du support de LinkBack;
  • correction des boutons inactifs pour appliquer les préambules à la bibliothèque;
  • amélioration de quelques points d'interface et correction de bugs mineurs.

Version 2.0.0 (4 novembre 2009)

  • revue du code avec de nombreux changements internes pour une meilleure flexibilité;
  • ajout des "modèles de corps";
  • historique et bibliothèque sont maintenant gérées par CoreData, ce qui améliore la gestion des ressources (mémoire...);
  • on peut maintenant donner des arguments supplémentaires aux programmes sous-jacents (pdlatex...);
  • les fenêtres de documents de LaTeXiT ont maintenant un mode "mini fenêtre" comme iTunes;
  • restauration de la compatibilité avec les processeurs G3;
  • amélioration du temps de démarrage;
  • correction de bugs dans les services;
  • correction de bugs mineurs;
  • traduction en Japonais;
  • le développeur lui-même est passé en version 2.0 en épousant sa femme bien aimée. Les dons sont bienvenus pour participer au voyage de noces !

Version 1.16.1

  • Meilleure compatibilité avec des versions anciennes de pdflatex/ghostscript;
  • correction d'un bug dans l'action Automator, qui l'empêchait d'être sauvegardée;
  • correction d'un bug dans la propagation des variables d'environnement pour les scripts personnalisés.

Version 1.16.0

  • Ajout d'une action Automator;
  • meilleure gestion de l'environnement Unix, qui corrige un bug rencontré avec certains configurations;
  • gestion de plusieurs préambules;
  • ajout de la palette "Fichiers supplémentaires";
  • amélioration du découpage automatique des équations;
  • gestion de plusieurs préambules;
  • pour la lisibilité, les icônes des fichiers créés n'ont plus un fond transparent;
  • la détection automatique de la taille des polices par le service est maintenant dotée d'un facteur multiplicatif paramétrable;
  • mise à jour de la traduction Allemande (Peter Dyballa);
  • correction de bugs mineurs.

Version 1.15.0

  • la version pour MacOS 10.3 (Panther) ne sera plus maintenue
  • correction d'une incompatibilité avec le shell de Leopard
  • les services fonctionnent à nouveau sous Leopard
  • gestion plus robuste des raccourcis claviers pour les Services
  • le service "Détecter et créer les équations" comprend maintenant \begin{eqnarray*}...\end{eqnarray*}
  • on peut maintenant importer une bibliothèque provenant de LaTeX Equation Editor
  • les tabulations ne sont plus arbitrairement limitées en nombre
  • LaTeXiT utilise maintenant Sparkle pour les mises à jour
  • la latexisation utilise maintenant le raccourci ⌘T comme TeXShop; ⌘L, ⇧⌘L, ⇧⌘T fonctionnent encore
  • le préambule par défaut utilise maintenant \usepackage[usenames]{color} au lieu de \usepackage{color}
  • nouvelle icône de Jacob Bandes-Storch
